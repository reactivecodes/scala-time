Scala Time

Scala Time is a simple, wrapper for `java.time` time APIs available in JDK versions 1.8 (Java 8) and greater.

Documentation

Usage

This project is not yet considered stable, and the API is subject to change, however binaries are published to the Maven Central Repository.

If using sbt, the current artefacts can be found at the following coordinates:

"codes.reactive" %% "scala-time" % "0.4.0"


See version specific documentation for usage examples and information.
